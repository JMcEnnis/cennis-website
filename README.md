# CENNIS Capital Advisory Website

A Next.js + Tailwind public website and starter app structure for CENNIS Capital Advisory.

## Pages

- `/`
- `/about`
- `/services`
- `/pricing`
- `/resources`
- `/contact`
- `/disclosures`

## Future App Modules

- `/dashboard`
- `/client-portal`
- `/onboarding`
- `/research`
- `/portfolio-analytics`

## Run Locally

Install dependencies, then start the dev server:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
